# MCP 2025-11-25 Compliance Matrix

This matrix tracks `infra-lens-mcp` against the latest Model Context Protocol specification version available on 2026-06-24: **2025-11-25**.

Status legend:

- **Supported**: implemented and covered by existing tests or release checks.
- **Partial**: implemented for the current scope, but missing spec behavior, tests, or production hardening.
- **Delegated**: intentionally handled by an external gateway, client, or deployment control.
- **Unsupported**: not implemented in this package.

## Current publication stance

The npm package artifact can be internally consistent while public connector publication remains blocked. `pnpm run check:metadata` therefore reports two separate concepts:

- `packageReady: true` for package, manifest, repository, file-list, and optional `dist` consistency.
- `connectorPublishReady: false` with an explicit `publishBlocker` while production OAuth/HTTPS gateway requirements remain outside this package.

This is intentional. Do not change `connector_readiness.publishReady` to `true` until #48 is resolved and verified.
